为什么保存任务后更新任务数组

Remult 的 Repository.save 方法发出一个 PUT 或一个 POST 请求，这取决于对象中是否存在一个 id 值 Task。

在本教程的下一节中，我们将通过创建 Task 对象并使用相同的 saveTask 函数保存它们来将新任务添加到列表中。因此，为了确保新创建的任务只被 POST-ed 一次，我们必须将其替换为 的返回值 Repository.save，其中包含一个 id.
